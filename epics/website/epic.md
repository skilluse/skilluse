# Website Epic

SkillUse Website - The official website for the SkillUse project at agentskills.io.

## Overview

A Next.js-based website providing documentation, blog posts, and landing pages for the SkillUse CLI tool and the Agent Skills Specification.

## Key Features

1. **Documentation** - Comprehensive guides for using the CLI and creating skills
2. **Blog** - Technical articles and updates about AI agent skills
3. **Landing Pages** - Product marketing pages for the CLI tool
4. **Responsive Design** - Mobile-friendly typography and layout
