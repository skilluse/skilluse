## What is SkillUse?

SkillUse is a CLI tool for managing AI agent skills. It helps you:

- **Discover** skills from GitHub repositories
- **Install** skills for Claude Code, Codex CLI, and other AI assistants
- **Manage** your installed skills with easy upgrade and removal
