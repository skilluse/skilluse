## Introduction

SkillUse is a powerful CLI tool that helps you discover, install, and manage skills for AI coding assistants like Claude Code, Codex CLI, and more.

## Installation

Install SkillUse globally using npm:

```bash
npm install -g skilluse
```

Or using bun:

```bash
bun install -g skilluse
```

## Quick Start

1. **Login to GitHub**:
   ```bash
   skilluse login
   ```

2. **Add a skill repository**:
   ```bash
   skilluse repo add owner/repo
   ```

3. **Install a skill**:
   ```bash
   skilluse install skill-name
   ```
